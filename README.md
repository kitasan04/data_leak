# data_leak
## 概要
　以下のことが出来ます。
- データ操作
  - データをリークさせる
  - データを元の状態に戻す
- データ確認
  - リークしたデータを確認する
  - 現在のデータ数を確認する
  - 元の状態のデータ数を確認する
## セットアップ
1. データを以下のフォーマットで用意する。<br>
data<br>
　├── test/<br>
　└── train/<br>
 <br>
2. ディレクトリパスを書き換える<br>
　なお、trainとtestをdata_leak下に置く場合は、書き換える必要はない。<br>
<br>
3. main.pyを実行し、1を入力し、0を入力する。その後yesを入力する。

## 使い方
main.pyを実行する
